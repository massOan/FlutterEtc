### 📝 コミットメッセージのタイプ一覧

| タイプ        | 使用タイミング                                                      |
|---------------|---------------------------------------------------------------------|
| feat          | 新しい機能の追加                                                   |
| fix           | バグ修正                                                           |
| docs          | ドキュメントの修正（READMEなど）                                   |
| style         | コードのスタイル修正（フォーマット、セミコロン追加など）機能の変更なし |
| design        | UI/UXデザインの変更（CSSなど）                                     |
| test          | テストコードの追加・変更                                           |
| refactor      | コードのリファクタリング（機能変更なし）                           |
| build         | ビルド関連の修正（build.gradleなど）                               |
| ci            | CI（継続的インテグレーション）設定ファイルの変更                    |
| perf          | パフォーマンス改善                                                 |
| chore         | 雑務（ビルド設定、パッケージ更新、Gitignoreの修正など）            |
| rename        | ファイル名・フォルダ名の変更のみを行った場合                        |
| remove        | ファイル削除のみを行った場合                                        |
